# GIF Maker Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task.

**Goal:** 在视频页添加"制作 GIF"功能,用户从当前播放位置起,选取 3-8 秒的视频片段生成 GIF 动画并自动下载。

**Architecture:**
- 前端 Plyr `currentTime` 拿到起点,POST 到 `/api/gif/make`
- 后端 ffmpeg 双 pass:`palettegen` 调色板 → `paletteuse` 编码 GIF
- 流式返回 GIF 字节给浏览器
- 前端 `fetch` 拿 Blob → `URL.createObjectURL` → `<a download>.click()`

**Tech Stack:** Express + ffmpeg (child_process) + Vue 3 + Plyr + Jest + supertest

**User-confirmed defaults (2026-06-21):**
- 帧率 12 fps
- 输出宽度 480px 等比缩放
- 视频剩余时长 < 选择时长时自动截到末尾(响应头 `X-Actual-Duration` 告知前端)
- 进度提示:按钮显示 loading,不上 WebSocket

## Architecture Diagram

```
VideoView.vue
   |
   | user clicks "制作 GIF" button
   v
Plyr.currentTime → startTime (seconds)
   |
   | popover: duration slider 3-8s, default 3
   v
fetch POST /api/gif/make
   body: { videoPath, startTime, duration }
   |
   v
server/routes/gif.js
   1. 校验 videoPath via isPathAllowed
   2. checkFFmpeg (reuse scanner/spriteGenerator)
   3. getVideoDuration → endTime = min(startTime + duration, total)
   4. mkdir tempDir/{taskId}
   5. ffmpeg pass1: palettegen → tempDir/{taskId}/palette.png
   6. ffmpeg pass2: paletteuse → tempDir/{taskId}/out.gif
   7. res.setHeader('X-Actual-Duration', String(actualDuration))
   8. fs.createReadStream(out.gif).pipe(res)
   9. res.on('close', () => rm -rf tempDir/{taskId})
   |
   v
Browser
   response.blob() → URL.createObjectURL(blob)
   → <a download="video_start3s.gif">.click()
   → revokeObjectURL
```

### Step 1.1: 写路由

`server/routes/gif.js` 实现:

```js
const express = require('express');
const path = require('path');
const fs = require('fs');
const os = require('os');
const crypto = require('crypto');
const { execFile } = require('child_process');
const { isPathAllowed } = require('../middleware/path');
const { checkFFmpeg, getVideoDuration } = require('../../scanner/spriteGenerator');

const router = express.Router();

const FPS = 12;
const WIDTH = 480;
const MAX_DURATION = 8;
const MIN_DURATION = 3;
const DEFAULT_DURATION = 3;
const TIMEOUT_MS = 60000;

function sanitizePath(input) {
  if (typeof input !== 'string' || !input) return null;
  try {
    return path.resolve(input);
  } catch {
    return null;
  }
}

router.post('/make', async (req, res) => {
  const { videoPath, startTime, duration } = req.body || {};

  // 1. 校验 videoPath
  const resolved = sanitizePath(videoPath);
  if (!resolved) {
    return res.status(400).json({ error: '缺少 videoPath 参数' });
  }
  if (!isPathAllowed(resolved)) {
    return res.status(403).json({ error: '禁止访问该路径' });
  }
  if (!fs.existsSync(resolved)) {
    return res.status(404).json({ error: '视频文件不存在' });
  }

  // 2. 校验时长参数
  const reqDuration = typeof duration === 'number' && duration >= MIN_DURATION && duration <= MAX_DURATION
    ? duration
    : DEFAULT_DURATION;
  const start = typeof startTime === 'number' && startTime >= 0 ? startTime : 0;

  // 3. 校验 ffmpeg
  let ffmpegPath;
  try {
    ffmpegPath = checkFFmpeg();
  } catch (err) {
    return res.status(500).json({ error: 'ffmpeg 不可用' });
  }

  // 4. 算实际 endTime
  let totalDuration;
  try {
    totalDuration = await getVideoDuration(resolved, ffmpegPath);
  } catch (err) {
    return res.status(500).json({ error: '无法获取视频时长' });
  }

  const endTime = Math.min(start + reqDuration, totalDuration);
  const actualDuration = endTime - start;
  if (actualDuration <= 0) {
    return res.status(400).json({ error: '起点已超出视频末尾' });
  }

  // 5. 准备临时目录
  const taskId = crypto.randomBytes(8).toString('hex');
  const tempDir = path.join(os.tmpdir(), `gif-maker-${taskId}`);
  fs.mkdirSync(tempDir, { recursive: true });
  const palettePath = path.join(tempDir, 'palette.png');
  const gifPath = path.join(tempDir, 'out.gif');

  const cleanup = () => {
    try {
      fs.rmSync(tempDir, { recursive: true, force: true });
    } catch { /* 忽略 */ }
  };

  res.on('close', cleanup);

  // 6. Pass 1: palettegen
  const paletteArgs = [
    '-y',
    '-ss', String(start),
    '-t', String(actualDuration),
    '-i', resolved,
    '-vf', `fps=${FPS},scale=${WIDTH}:-1:flags=lanczos,palettegen`,
    palettePath,
  ];

  execFile(ffmpegPath, paletteArgs, { timeout: TIMEOUT_MS }, (err) => {
    if (err) {
      cleanup();
      if (!res.headersSent) {
        return res.status(500).json({ error: 'palettegen 失败: ' + err.message });
      }
      return res.end();
    }

    // 7. Pass 2: paletteuse
    const gifArgs = [
      '-y',
      '-ss', String(start),
      '-t', String(actualDuration),
      '-i', resolved,
      '-i', palettePath,
      '-filter_complex', `[0:v]fps=${FPS},scale=${WIDTH}:-1:flags=lanczos[x];[x][1:v]paletteuse`,
      gifPath,
    ];

    execFile(ffmpegPath, gifArgs, { timeout: TIMEOUT_MS }, (err) => {
      if (err) {
        cleanup();
        if (!res.headersSent) {
          return res.status(500).json({ error: 'paletteuse 失败: ' + err.message });
        }
        return res.end();
      }

      // 8. 流式响应
      if (!fs.existsSync(gifPath)) {
        cleanup();
        if (!res.headersSent) return res.status(500).json({ error: 'GIF 生成失败' });
        return res.end();
      }

      const stat = fs.statSync(gifPath);
      res.setHeader('Content-Type', 'image/gif');
      res.setHeader('Content-Length', String(stat.size));
      res.setHeader('X-Actual-Duration', String(actualDuration));
      res.setHeader('Content-Disposition', `attachment; filename="clip_${Math.floor(start)}s.gif"`);

      const stream = fs.createReadStream(gifPath);
      stream.pipe(res);
      stream.on('error', () => {
        if (!res.headersSent) res.status(500).end();
        cleanup();
      });
    });
  });
});

module.exports = router;
```

async function makeGif(params: { videoPath: string; startTime: number; duration: number }): Promise<Blob> {
  const res = await fetch('/api/gif/make', {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },

  })
  if (!res.ok) {
    const err = await res.json().catch(() => ({ error: res.statusText }))

  return res.blob()

## Risks

| Risk | Severity | Mitigation |
|------|----------|------------|
| 8s 视频双 pass ffmpeg 慢(可能 10s+) | MEDIUM | 60s 硬超时;用户默认 3s 通常 2-3s |
| GIF 文件大(8s 约 1.5-3MB) | LOW | 480p 12fps 已平衡;后续可调 |
| 视频总时长 < 3s | LOW | 自动截断 + 头告知,前端 alert |
| popover 被 Plyr 控件遮挡 | MEDIUM | z-20 + 实测调整 |
| 并发请求临时目录冲突 | LOW | 每个请求用独立 taskId 目录 |
| 临时文件残留(进程被杀) | LOW | res.on('close') + 进程退出兜底复用 frameExtractor 模式 |
